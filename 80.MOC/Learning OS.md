# 🧠 Learning OS · 学习操作系统

> **核心信念：** 输入是原料，输出才是复利。每天一个输入 × 一个输出 = 知识的复利积累。
> 
> **你已经有的：** 3160条flomo笔记 · 16篇学术文献 · 10个设计项目案例 · 90+条永久笔记
> **你缺的：** 每天把这些变成自己思想的流水线

---

## 📐 系统架构：三层复利引擎

```
原料层（每天挖）         加工层（每天写）         产出层（每周串）
──────────────         ──────────────         ──────────────
flomo 3160条           每日永久笔记             每周综述
学术文献 L1-L16    →   连接旧知识        →      文章/输出
设计案例 P1-P10        提出新问题               IRP观点
外部文章/世界           回答：改变了什么？         个人思想体系
```

## ⏱️ 每日仪式（50分钟 · 每天固定时间）

### Step 1 · MINE · 挖矿（5分钟）
打开一条 flomo 旧笔记，问自己：
- "过去的我为什么记这个？"
- "现在的我怎么看这件事？"

> 🎲 随机挖：在搜索栏输入任意一个词（如"复利"、"选择"、"食物"），看跳出什么

---

### Step 2 · READ · 深读（25分钟）
每天读**一篇**，不是浏览，是深读。来源按优先级：

**本周优先（IRP相关）：**
- [ ] [[50.Knowledge/Literature/Academic/L7 - Kahneman (2011)]] · 思考快与慢
- [ ] [[50.Knowledge/Literature/Academic/L9 - Thaler and Sunstein (2008)]] · 助推理论
- [ ] [[50.Knowledge/Literature/Academic/L1 - Fogg (2009)]] · 行为设计
- [ ] [[50.Knowledge/Literature/Academic/L13 - Rozin et al (1999)]] · 食物与情绪

**通用深度阅读来源：**
- Aeon Magazine (aeon.co) · 每篇约15分钟
- 少数派 / 得到 · 中文深度内容
- 你的 flomo 里收藏的任意链接

**读的时候只问三个问题：**
1. 核心主张是什么？（一句话）
2. 证据是什么？（反例是什么？）
3. 和我已知的什么连接？

---

### Step 3 · WRITE · 炼金（20分钟）
写一条**永久笔记**，放进 `50.Knowledge/Permanent/`

**模板很简单（不需要完美）：**

```
# [今天的核心洞见，用自己的话说]

## 触发点
今天读了/想到了什么

## 核心观点
用自己的话，一段话说清楚

## 连接
和哪些已有想法有关？
→ [[已有笔记]]

## 改变了什么
这个想法改变了我的什么认知？或者引出了什么新问题？
```

> ⚡ 关键：**"改变了什么"这一栏不能空着。** 空着就是白读。

---

## 📅 每周仪式（周日60分钟）

### 串联这周的笔记
打开这周写的所有永久笔记，找到：
- 重复出现的词/主题
- 互相矛盾的观点
- 可以合并成一个更大观点的碎片

然后写一篇**周综述**，放进 `10.Daily/周复盘/`

格式：
```
本周读了什么 → 本周想明白了什么 → 下周要验证什么
```

## 💡 防内耗守则

1. **今天只做一件事**：读完一篇 + 写完一条笔记
2. **不完美也要完成**：笔记写三句话也算完成
3. **配置到此为止**：系统已经够用，开始用它
4. **进度不靠感觉靠记录**：每天写了就有，没写就没有
