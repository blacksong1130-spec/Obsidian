### 📄 PDF 类

#### `/pdf`
| 项目 | 内容 |
|------|------|
| **触发方式** | 说"处理 PDF"、"合并/拆分/创建 PDF"，或提到 `.pdf` 文件 |
| **目的** | 处理一切 PDF 相关任务 |
| **实现效果** | 合并多个 PDF、拆分单页、提取文字/表格/图片、加水印、加密/解密、OCR 扫描件转文字、填写 PDF 表单、创建新 PDF |

#### `/pdf-extraction`
| 项目 | 内容 |
|------|------|
| **触发方式** | 说"提取 PDF 里的文字/表格"、"解析这份报告" |
| **目的** | 精准提取 PDF 中的结构化数据 |
| **实现效果** | 用 pdfplumber 提取文字（保留排版）、提取表格并转为 Excel/DataFrame、按区域/字体/字号过滤内容、可视化调试 PDF 结构 |

### ⚙️ 系统与工具类

#### `/huashu-nuwa`（女娲造人）
| 项目 | 内容 |
|------|------|
| **触发方式** | "造 skill"、"蒸馏XX"、"女娲"、"做个XX视角的 skill"、"我想要一个XX的思维顾问" |

| **目的** | 从 Skills 生态中发现并安装新 Skill |
