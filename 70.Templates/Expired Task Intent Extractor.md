# 🗑️ 过期任务意图提取器 (Expired Task Intent Extractor)

> **知识大师心法**：
> 任务有保质期，但产生任务的“意图”没有。
> 永远不要因为内疚而去完成一个已经过期的任务，也不要盲目同步回 Inbox。
> 我们要做的是：**干掉任务，提取意图。**

## 2. 意图审判 (Triage)
*(问自己：当时为什么记下这个任务？它背后隐藏了我怎样的长期渴望或焦虑？)*

- **底层意图**：(如：其实不是缺服务器，而是当时想低成本尝试建站副业)
- **知识折旧率**：[ ] 彻底失效 (直接删除) / [ ] 具有长半衰期价值 (进入下一步)
