## 一、面部皮肤分析

### 皮肤类型：油性 / 混合性

根据照片观察，皮肤存在以下问题：

| 问题 | 具体表现 | 严重程度 |
|------|---------|---------|
| 皮肤出油 | T区（额头、鼻子、两颊）明显油光发亮 | 中度 |
| 活跃炎症型痘痘 | 两颊可见红色丘疹 / 脓疱型痘痘 | 中度 |
| 痘印 / 炎症后色素沉着 | 旧痘留下的红色及暗色印记 | 轻中度 |
| 毛孔粗大 | 油性皮肤造成的毛孔明显 | 轻度 |
| 肤色不均 | 因痘痘及痘印导致局部发红、色差明显 | 轻度 |

## 二、头发状况分析

| 问题 | 具体表现 | 严重程度 |
|------|---------|---------|
| 毛躁 / 飞发 | 发顶及发尾有明显炸毛，整体蓬乱 | 中度 |
| 卷曲无定型 | 自然波浪卷曲但没有形状和光泽 | 中度 |
| 发尾干燥 | 发丝缺乏水分，发尾无光泽 | 中度 |
| 整体造型凌乱 | 发型缺乏层次感与控制感 | 轻度 |

## 三、专业护理方案

### 护肤日程（每日）

```
早上
  1. 水杨酸泡沫洁面
  2. 烟酰胺 + 锌精华（全脸）
  3. 水杨酸 + 烟酰胺凝胶保湿霜
  4. SPF50+ 防晒（必须）

晚上
  1. 水杨酸泡沫洁面
  2. 烟酰胺 + 锌精华（全脸）
  3. 水杨酸 + 烟酰胺凝胶保湿霜
  4. Acnecide（仅点涂有痘痘处，每周 2-3 次）
```

### 护发日程（每次洗发）

```
1. OGX 阿甘油洗发水（正常洗发，揉搓头皮）
2. OGX 阿甘油护发素（重点涂发尾，停留 3-5 分钟）
3. 吹至半干后，取几滴 OGX 阿甘油渗透发油涂于发尾
```

## 四、成分选择逻辑

| 成分 | 作用 | 针对问题 |
|------|------|---------|
| 水杨酸（Salicylic Acid）| 溶解堵塞毛孔的油脂和角质，减少黑头与粉刺 | 出油、痘痘、毛孔 |
| 烟酰胺（Niacinamide）| 控油、消炎、淡化痘印、收缩毛孔 | 痘印、油光、肤色不均 |
| 锌（Zinc）| 抗炎，辅助控油 | 活跃痘痘、红肿 |
| 苯甲酰过氧化物（BPO 5%）| 直接杀灭痘痘细菌，临床级别最强非处方成分 | 活跃炎症痘痘 |
| SPF50+ 防晒 | 防止 UV 照射让痘印变深变黑，是淡印最关键步骤 | 痘印加深 |
| 摩洛哥阿甘油 | 深层滋润、修护受损发丝，控制毛躁增加光泽 | 毛躁、干燥发尾 |
