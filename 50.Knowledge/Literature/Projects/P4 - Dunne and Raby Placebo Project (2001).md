# P4 — Dunne & Raby (2001): Placebo Project

## 📖 Citation (Harvard)
Dunne, A. and Raby, F. (2001) *Placebo Project* [Critical design project]. London: Royal College of Art. Available at: https://www.dunneandraby.co.uk/content/projects/69/0 (Accessed: 6 April 2026).

---

## What / So What / Now What

**What:** The Placebo Project created eight speculative electronic objects using the body's electromagnetic environment as an interface: an object that twitched detecting mobile phone radiation, a chair vibrating with the body's own electrical field, a pillow displaying the owner's heartbeat. These were not functional products but design provocations asking what our relationship with invisible technology might feel like if we could sense it.

**So What:** Direct precedent for this project's speculative approach: using artefacts to make **invisible biological signals perceivable**. The parallel is precise — metabolic processes are as invisible as electromagnetic fields, and the design challenge is similar: how do you translate subcellular biology into a meaningful human experience without medicalising it?

**Now What:** Use as direct methodological precedent for the speculative prototyping phase. The objects produced in this project should aim for the same quality: not functional prototypes but **provocative artefacts** that ask the right questions. Connects to [[L2 - Dunne and Raby (2013)]] for theoretical grounding.

## 🔑 Key Design Qualities & Lessons

| Placebo Project Quality | Why It Matters | Application to This Project |
|------------------------|---------------|----------------------------|
| **Making invisible visible** | Gives tangible form to imperceptible signals | Metabolic biology → perceivable feedback |
| **Non-medical framing** | Not a diagnostic tool — an experiential one | Dietary feedback should feel like art, not a blood test |
| **Ambiguity** | Objects don't prescribe meaning; users interpret | Feedback that invites reflection, not instruction |
| **Body as interface** | Uses the body's own signals as data | Gut feeling, skin response, warmth as feedback channels |
| **Provocative not prescriptive** | Opens questions rather than answering them | Speculative prototypes = provocations, not products |

| Overall Assessment | Detail |
|---|---|
| **Key Strength** | Shows that making invisible biology tangible is designable and culturally resonant |
| **Key Limitation** | Not user-tested for behaviour change; remains in gallery/fiction space |
| **Design Lesson** | The most powerful feedback might not be digital — texture, warmth, sound, smell |
