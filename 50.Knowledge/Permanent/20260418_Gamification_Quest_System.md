# 20260418 · 任务结算系统 · 游戏化知识大陆

## 系统架构（高维设计）

这套系统的底层逻辑来自你的 IRP 研究本身：
**Fogg 行为模型 + 立即反馈机制**

```
行为发生 = 动机 × 能力 × 触发器
                        ↑
              结算仪式就是这个触发器
              它在行为完成时立即响应
              让大脑把「完成」和「奖励」绑定
```

上瘾回路（来自 Nir Eyal《Hooked》）：
```
触发 → 行动 → 可变奖励 → 投入
 ↑                         ↓
 └─────────────────────────┘
        下一个循环
```

「可变奖励」是关键：固定奖励很快就无聊，
随机的幸运加成和随机名言制造了不确定性，
不确定性让大脑保持期待。
